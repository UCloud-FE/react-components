### 说明

-   这是 Checkbox Group 组件，用于包裹 Checkbox 组件并获取数据
-   也可以直接使用 options 来定义选项
-   使用 group 时请使用 value 或 defaultValue 来控制 checkbox 的勾选，不要使用 checked 或 defaultChecked
-   初始值请注意保证准确性防止出现意外的情况
