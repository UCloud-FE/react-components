### 说明

-   这是 EditableTable 组件，主要用于列表项的编辑删除
-   与表格的区别
    1. pagination 固定为 null
    2. emptyContent 固定为 null
    3. columns 中添加了删除列
-   其余属性与 Table 保持一致
